2 Conceptos básicos

La Estadística nos permite identificar patrones, probar hipótesis o generar conclusiones desde conjuntos de datos donde la variación aleatoria prevalece. El insumo de la Estadística recae sobre aspectos como variables, población, muestra, etc. En seguida se dá una descripción de tales aspectos.

2.1 Variables

Una variable es una característica o atributo común a un conjunto de elementos de interés (p.e., personas, un arboles, caja de petri, etc.) y que toma valores diferentes de un elemento a otro. Las variables se pueden clasificar desde diferentes puntos de vista. Aquí algunos.

2.1.1 Variables por el tipo de valor que toman

2.2 Población y muestra

La población incluye el conjunto completo de elementos sobre el cuál se quiere hacer inferencia. La muestra es un grupo elemetos que aún.
